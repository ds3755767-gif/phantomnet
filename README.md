# 🔥 PhantomNet AI Defense

## 🚀 Project Overview
PhantomNet AI is an **Invisible Cyber Defense System for Banks**.  
Instead of detecting attacks after they happen, PhantomNet confuses attackers before they succeed.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Django REST Framework
- **AI Layer:** Scikit-Learn (Decision Tree Classifier)
- **Compliance:** MITRE ATT&CK Mapping (T1083, T1041)

## ⚙️ How It Works (Step-by-Step)
1. **Intruder Footprint Capture** → IP, User-Agent, Cyber DNA ID
2. **Dynamic Deception Shell Rotation** → Fake banking portal + honeypot
3. **AI Risk Scoring Engine (UEBA)** → Risk % increases with attacker actions
4. **Reverse Tar-Pit Core** → Delays attacker scripts
5. **Automated Isolation Short-Circuit** → ACCESS DENIED screen
6. **SIEM Compliance Intelligence** → Admin dashboard with MITRE mapping

## 🎯 Demo Flow
- Normal user → Real banking dashboard
- Attacker → Fake dashboard + logs recorded
- Admin → Live telemetry + exportable forensic report

## 📊 Architecture
- Frontend Layer → Trapping & Monitoring UI
- Backend Layer → Logic & Controls Engine
- AI Layer → Behavior analysis + compliance mapping
